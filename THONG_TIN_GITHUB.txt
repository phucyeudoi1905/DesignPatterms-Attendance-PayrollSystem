   cd AttendancePayrollSystem

   📌 Bước 2: Chạy Backend

   mvn spring-boot:run

   - Run AttendancePayrollSystemApplication.java

   cd frontend
   npm install
   npm run dev
   
   Frontend: http://localhost:5173

✨ TÍNH NĂNG CHÍNH

   1. Dashboard - Thống kê real-time, tự động refresh 30s
   2. Quản lý Nhân viên - CRUD, 4 loại hợp đồng
   3. Chấm công - Ghi nhận giờ làm/sản phẩm
   4. Tính lương - Tự động tính theo Strategy Pattern

🎨 DESIGN PATTERNS CHI TIẾT

   📊 Strategy Pattern (Behavioral)
   - Interface: SalaryCalculationStrategy
   - Concrete Strategies:
     • FixedSalaryCalculation
     • HourlySalaryCalculation
     • ProductBasedSalaryCalculation
     • KpiBasedSalaryCalculation
   - Context: PayrollService
   - Lợi ích: Dễ mở rộng, tuân thủ Open/Closed Principle

   🔧 Singleton Pattern (Creational)
   - Class: ApplicationConfig
   - Thread-safe implementation
   - Quản lý cấu hình toàn cục
   - Lợi ích: Single instance, truy cập toàn cục

   🎁 Decorator Pattern (Structural)
   - Component: SalaryComponent
   - Base: BaseSalary
   - Decorators:
     • TransportationAllowanceDecorator
     • LunchAllowanceDecorator
     • PerformanceBonusDecorator
     • OvertimeDecorator
   - Lợi ích: Thêm chức năng động, không sửa code gốc

🔗 API ENDPOINTS

   Base URL: http://localhost:8080/api

   Dashboard:
   GET  /dashboard/stats      - Thống kê tổng quan
   GET  /dashboard/config     - Cấu hình hệ thống

   Employees:
   GET    /employees          - Danh sách nhân viên
   GET    /employees/{id}     - Chi tiết nhân viên
   POST   /employees          - Thêm nhân viên
   PUT    /employees/{id}     - Cập nhật nhân viên
   DELETE /employees/{id}     - Xóa nhân viên

   Attendance:
   GET  /attendance/{employeeId}  - Lịch sử chấm công
   POST /attendance                - Thêm chấm công

   Payroll:
   POST /payroll/calculate/{employeeId}  - Tính lương
   GET  /payroll/{employeeId}            - Lịch sử lương
